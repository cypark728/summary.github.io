# git

## git branch <name>
  <name> 이란 이름의 branch를 만들 수 있습니다. 
    
## git branch
브랜치의 목록 전체를 확인 할 수 있습니다. 현재 선택된 브랜치는 앞에 * 표시되어 있습니다. 

## git checkout
마지막으로 commit 한 버전으로 돌아간다.

## git log 
지금까지 git 을 사용한 로그를 확인할 수 있다. 

## git reset
시간을 commit하기 전으로 돌린다. 

## git revert
해당 commit의 상태를 되돌린다. 

## git rebase
commit들의 base를 과거로 바꾼다. 

## git clone
해당 git의 내용을 복제한다.
